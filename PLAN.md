# PLAN.md

## 第一版目標

建立可離線使用的小學生注音、認字與語音學習 PWA，適合 iPhone 直式單題操作，並支援 iPad 直式與橫式自適應版面。

## 第一版範圍

1. 首頁
2. 注音符號學習卡
3. 點擊播放聲音
4. 聽聲音選注音
5. 看圖認字
6. 答對獲得星星
7. 學習紀錄儲存在本機
8. 家長模式入口
9. `manifest.json`
10. Service Worker
11. `README.md`

## 設計原則

- 單題呈現，減少兒童認知負擔。
- 按鈕大、間距大、文字清楚。
- 使用 `click` 事件處理觸控，不依賴滑鼠 hover。
- 聲音使用真人錄音優先；沒有音檔時使用 Web Speech API 朗讀。
- 不連外傳送學習紀錄。

## 後續版本方向

- 加入完整注音聲母、韻母、結合韻與聲調練習。
- 補上真人錄音檔與圖片素材。
- 加入更多常見國字、詞語與句子。
- 家長模式增加匯出本機紀錄與重置指定項目。
